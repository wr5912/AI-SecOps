# CyberSentinel - 企业智能安全运营平台

## 1. 项目概述

**项目名称**: CyberSentinel SOC Platform
**项目类型**: 企业级安全运营中心 Dashboard Web应用
**核心功能**: 用于监控、检测、分析和响应企业网络安全威胁的统一"单一窗口"平台，集成传统SIEM能力与AI驱动决策支持和攻击可视化
**目标用户**: SOC分析师（1-3级）、事件响应人员、CISO、安全工程师

## 2. UI/UX 规格

### 设计理念
- **主题**: 深色模式默认（降低24/7监控室眼睛疲劳）
- **美学风格**: "网络物理"风格 - 清晰、高对比度数据可视化、HUD元素、最小干扰
- **密度**: 高数据密度，可折叠侧边栏最大化工作空间

### 布局结构
- **全局导航（左侧）**: 基于图标的可折叠菜单，主要模块（监控、资产、情报、报告）
- **全局顶部栏**:
  - 全局搜索（IP、域名、哈希、工单ID）
  - 系统健康状态（CPU/RAM/网络负载）
  - "红色警报"滚动条（未确认的关键威胁）
  - 用户资料和角色切换
- **主工作区**: 12列响应式网格系统，允许小部件自定义
- **上下文面板（右侧-滑出）**: AI助手、快速操作或所选项目的详情视图

### 视觉设计
- **配色方案（深色模式）**:
  - 背景: `#0F172A` (深蓝/板岩色)
  - 表面/卡片: `#1E293B` (浅板岩色)
  - 主要操作: `#3B82F6` (科技蓝)
  - 关键/攻击: `#EF4444` (霓虹红)
  - 警告/可疑: `#F59E0B` (琥珀色)
  - 安全/正常: `#10B981` (翡翠绿)
  - 文本: `#F1F5F9` (主要), `#94A3B8` (次要)
- **字体**:
  - UI元素: Inter 或 Roboto（清晰无衬线）
  - 数据/代码/日志: JetBrains Mono 或 Fira Code（等宽字体对IP地址和日志必不可少）
- **视觉效果**:
  - 关键警报上的微妙发光效果 (`box-shadow`)
  - 叠加面板的毛玻璃效果 (`backdrop-filter: blur`)
  - 数据更新的平滑过渡 (0.2s ease-out)

### 核心组件
1. **攻击地图小部件**: 基于WebGL的3D地球仪或2D拓扑图，显示攻击来源/目标
2. **数据表**: 高性能虚拟化日志表（排序、筛选、列切换）
3. **时间线轨道**: 攻击链的线性可视化
4. **终端/控制台**: 嵌入式命令行用于快速查询
5. **指标卡片**: 迷你图 + 大数字 + 趋势指示器

## 3. 功能模块规格

### 模块1: 实时安全监控（仪表板）
- 全球威胁级别（防御状态样式）
- 实时事件流（传入警报的可滚动日志）
- 流量热图
- 甜甜圈图显示警报严重程度分布

### 模块2: 攻击态势感知（可视化器）
- 拓扑视图: 显示攻击横向移动的网络节点可视化
- 杀伤链映射: 可视化当前威胁在网络杀伤链中的位置（侦察->投递->利用）
- 交互: 点击节点隔离流量；拖动旋转地图

### 模块3: 资产和漏洞管理
- 资产清单表（服务器、终端、物联网设备）
- 漏洞扫描器集成结果
- 每个资产的"风险评分"徽章
- 树形图按部门分组资产，按风险评分调整大小

### 模块4: 事件响应和工单
- 看板（新、调查中、遏制、已解决）
- 集成剧本（分析师的分步标准操作程序）
- 证据库（文件附件、PCAP文件）
- 流程: 警报 -> 点击"创建事件" -> 自动填充数据 -> 分配分析师

### 模块5: 威胁情报和预警
- 订阅源聚合器（外部来源如Geometric，内部历史）
- IOC（危害指标）搜索
- 关联引擎: 将内部日志与外部情报自动匹配

### 模块6: 合规管理和审计
- 合规仪表板（ISO 27001、GDPR状态）
- 自动报告生成器（PDF/Excel导出）
- 审计跟踪日志（系统中谁做了什么）
- 雷达图比较当前态势与要求的标准

### 模块7: 应急响应和演练
- "作战室"模式: 危机期间切换到简化的顶级视图
- 演练模拟器: 注入虚假攻击数据测试团队响应
- 一键隔离: 隔离 compromised子网的按钮（需要双因素认证）

### 模块8: AI智能和决策支持
- **安全副驾驶**: 聊天界面侧边栏（"分析这个IP地址"、"总结这个事件"）
- **根本原因分析**: AI自动为警报生成假设图
- **异常检测**: 突出显示即使没有规则触发也"异常"的行为
- **响应建议**: "AI建议基于98%置信度阻止端口445"

## 4. 交互体验优化
- **上下文操作**: 右键点击任何IP/哈希显示菜单："添加到黑名单"、"Whois查询"、"VirusTotal扫描"
- **键盘快捷键**: `cmd+k` 搜索，`[]` 循环事件，`esc` 关闭面板
- **下钻能力**: 点击图表片段立即筛选主数据表
- **可自定义仪表板**: 拖放小部件创建角色特定视图（如经理视图 vs 分析师视图）

## 5. 验收标准
1. **延迟**: 仪表板数据在后端事件后<200ms更新
2. **清晰度**: 关键警报（红色）必须可被色盲用户区分（使用图标+颜色）
3. **响应式**: 布局适应大型控制室屏幕（4K+）和标准笔记本电脑（1080p）
4. **导出**: 报告必须生成包含可视化图表的有效PDF/CSV文件
